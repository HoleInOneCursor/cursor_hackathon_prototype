## How agents run this repo

1. Confirm you are on the intended feature branch and that the working tree is clean.
2. Inspect the existing files before making changes.
3. Keep edits focused on the requested task.
4. Validate Markdown formatting for documentation-only changes.
5. Commit and push changes to the feature branch.

No install, build, lint, or test commands are currently required for this repository.
