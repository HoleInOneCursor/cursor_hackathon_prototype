# cursor_hackathon_prototype

## Agent run instructions

This repository is currently documentation-only. Agents should run from the
repository root and can edit `README.md` directly without installing
dependencies or starting a local service.

Before submitting changes, verify the README renders as expected and run:

```sh
git status
git diff -- README.md
```

There are no package-manager install steps, build commands, or automated tests
configured in this repository at this time.
