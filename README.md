# cursor_hackathon_prototype

## How agents run this repo

1. Start from the assigned feature branch and confirm it is up to date.
2. Inspect the working tree before editing so any existing changes are understood.
3. Read the relevant files for context, then make focused edits.
4. Validate Markdown formatting for documentation changes.
5. Commit the changes, push the branch, and open or update the pull request.

No install, build, lint, or test commands are currently needed for this repository.
