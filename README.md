# cursor_hackathon_prototype

## How agents run this repo

This repository currently contains documentation only. There is no application
runtime, package manager configuration, build step, or automated test suite to
run yet.

When working in this repo, agents should:

1. Check the current branch and working tree with `git status`.
2. Read this README for the latest project and workflow notes.
3. Make documentation changes directly; no dependency installation is required.
4. Validate changes by reviewing the Markdown diff and running
   `git diff --check` before committing.

If future code or tooling is added, document the exact install, run, build, and
test commands in this section so agents have a single source of truth.
