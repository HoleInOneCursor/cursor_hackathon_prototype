# cursor_hackathon_prototype

## Running this repo with agents

This repository currently contains project documentation only. Agents working in
this repo should:

1. Start from the repository root.
2. Check `git status` before making changes.
3. Edit the documentation directly; there are no package installs, build steps,
   or automated tests to run at this time.
4. Validate documentation changes by reviewing the rendered Markdown or reading
   the updated file.
5. Commit and push changes on the active task branch before opening or updating
   a pull request.
