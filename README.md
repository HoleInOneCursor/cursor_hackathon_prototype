# cursor_hackathon_prototype

## How agents run this repo

This repository currently contains project documentation only. Cursor Cloud agents
should work from the assigned feature branch, keep changes focused, and submit
their work through a pull request.

Recommended agent workflow:

1. Confirm the current branch with `git status --branch`.
2. Review the repository contents before making changes.
3. Edit the relevant documentation or source files.
4. Run any applicable checks. At the moment, there are no build, lint, or test
   commands configured for this repo.
5. Commit changes with a descriptive message.
6. Push with `git push -u origin <branch-name>`.
7. Open or update a pull request against `main`.
