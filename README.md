# cursor_hackathon_prototype

## How agents run this repo

This repository currently contains documentation only. Agents working on this
repo should:

1. Check out the assigned feature branch.
2. Read this README for project context and task-specific guidance.
3. Make documentation changes directly in the repository.
4. Verify changes with `git diff` and `git status`.

There are no dependency installation, build, lint, or test commands required at
this time. If code or tooling is added later, update this section with the exact
commands agents should run before opening a pull request.
