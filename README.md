# cursor_hackathon_prototype

## Running this repo with agents

This repository currently contains documentation only. Agents working on this repo should:

1. Work from the assigned feature branch and keep changes scoped to the requested task.
2. No dependency installation, build step, or test suite is required until project source files are added.
3. Validate changes with `git status` and `git diff` before committing.
4. Commit with a clear message and push with `git push -u origin <branch-name>`.
5. Open a pull request that summarizes the change and notes any validation performed.
