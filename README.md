# cursor_hackathon_prototype

## How agents run this repo

This repository is currently documentation-only. Agents should treat the
top-level `README.md` as the primary project artifact until source code,
configuration, or test tooling is added.

Recommended agent workflow:

1. Start from the assigned feature branch and confirm the working tree is clean.
2. Inspect the repository before making changes so new instructions stay aligned
   with the files that actually exist.
3. Make focused edits in place; avoid adding build, package, or test tooling
   unless the task explicitly requires it.
4. Validate documentation changes by reviewing the rendered Markdown or reading
   the updated file for formatting issues.
5. Commit and push changes to the assigned branch, then open or update the pull
   request for review.

There are no install, build, lint, or test commands required for the current
state of this repo.
